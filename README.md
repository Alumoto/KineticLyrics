# KineticLyrics
KineticLyricsは、kra形式歌詞ファイルと音源ファイルから、Kinetic TypographyやLyric Speakerのような歌詞アニメーションを生成するソフトウェアです。タイムタグを利用し、音源と歌詞を同期させて表示します。

## 開発言語
ProcessingとJavaで開発されています。

## 現在の進捗
-歌詞表示機能
    -非同期表示(txt)
        -左から右へスクロール
        -上/下から現れてひっこむ
        -一文字ずつ登場、消える
    -同期表示(kra)
        -左から右へスクロール
-音源読み込み機能
-歌詞読み込み機能
    -txt、kraファイルに対応。

## 今後の展望

    -より複雑な表現を可能にする
